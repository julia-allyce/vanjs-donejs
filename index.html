VanJS, January 18, 2016

So what is it?

  • Open Source JavaScript framework (MIT)
  • Backend services agnostic
  • Windows, Linux, OSX
  • Comes with
    • StealJS - ES6, CJS, and AMD module loader and builder
    • CanJS - Custom elements and Model-View-ViewModel utilities
    • jQuery - DOM helpers
    • FuncUnit - Functional tests
    • Testee - QUnit, Mocha and Jasmine Test runner
    • DocumentJS - Documentation
    • can-ssr - Server-Side Rendering Utilities for CanJS
    • can-connect - Assemble real-time, high performance, restful data connections.

Server side Rendering

Real-time

npm install steal-socket.io --save

import io from 'steal-socket.io';

const socket = io('http://chat.donejs.com');

socket.on('messages created',
  message => messageConnection.createInstance(message));
socket.on('messages updated',
  message => messageConnection.updateInstance(message));
socket.on('messages removed',
  message => messageConnection.destroyInstance(message));
